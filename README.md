# 🐍 Devil’s Serpent — Multi-Level Snake Game (Java)

A structured, level-based Snake game built in Java that focuses on **game state management, progression systems, and real-time interaction**, going beyond a basic endless implementation.

---

## 🎯 Overview

This project started as a simple Snake game but evolved into a **multi-level system** with increasing difficulty, obstacles, and controlled progression.

Instead of an endless loop, the game introduces:

* defined levels
* structured difficulty scaling
* gameplay progression

The goal was to understand how real-time applications manage:

* continuous updates (game loop)
* user input handling
* state transitions across multiple stages

---

## 🚀 Features

* 🎮 **6 Progressive Levels**

  * Each level introduces new challenges and increased difficulty

* 🚧 **Obstacle System**

  * Static obstacles
  * Moving obstacles (dynamic difficulty)

* ⚡ **Speed Scaling**

  * Game speed increases as levels progress

* 🧠 **State-Based Gameplay**

  * Transition between menus, gameplay, and game-over states

* 🏆 **Score Tracking**

  * Tracks current score
  * Displays highest score achieved

* 📖 **Instruction & UI Screens**

  * Dedicated instruction pages
  * Level selection interface

---

## 🧠 Core Concepts

### 🔹 Game Development Concepts

* Game loop design (continuous updates)
* Event-driven input handling
* State management (menu → game → result)

---

### 🔹 System Design Thinking

* Breaking game into manageable components
* Handling transitions between levels
* Managing increasing complexity without breaking structure

---

### 🔹 GUI Programming

* Java Swing for interface
* Custom rendering and screen management
