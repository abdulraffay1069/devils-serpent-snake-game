# 🐍 Devil’s Serpent — Snake Game with Level Progression (Java)

A feature-rich Snake game built using Java, designed to explore game logic, state management, and real-time interaction beyond a basic implementation.

---

## 🎯 Overview

This project started as a simple Snake game but was extended into a multi-level system with increasing difficulty, obstacles, and structured gameplay.

The goal was to understand how real-time applications manage state, user input, and progression.

---

## 🚀 Features

* 🎮 Multiple levels (6 stages with increasing difficulty)
* 🚧 Static and moving obstacles
* ⚡ Speed increases as levels progress
* 🧠 Level-based gameplay instead of endless mode
* 🏆 Score tracking with highest score display
* 📖 Instruction and guidance screens
* 🎯 Clear progression system across levels

---

## 🧠 What This Project Focuses On

* Managing game loops and continuous updates
* Handling real-time user input
* Structuring a multi-level system
* Designing gameplay progression instead of single-state logic
